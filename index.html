視差捲動特效

作者：Vineet Kumar

什麼是視差捲動 (Parallax Scrolling)？

視差捲動是一種網頁設計技術，讓網頁背景的移動速度比前景慢。當訪客向下捲動頁面時，這會產生 3D 立體效果，增加景深感並創造更身歷其境的瀏覽體驗。 視差是基於視覺錯覺的原理。由於人眼會覺得靠近我們的物體比遠處的物體大，我們也會感知到遠處的物體移動得較慢。 這種錯覺早已被應用於各種媒體中以營造真實感。它最早用於傳統動畫，可追溯到迪士尼的《白雪公主與七個小矮人》，以及像《超級瑪利歐》這樣的經典電玩遊戲。 隨著 CSS 和 HTML 的進步，視差效果後來演變成了我們今天所熟知的網頁設計形式。

早在 2007 年，網頁開發者 Glutnix 就在部落格文章中分享了瀏覽器端視差捲動的早期實作，其中包含了範例程式碼以及使用 JavaScript 和 CSS 2 製作的 Demo，當時甚至還支援 Internet Explorer 6。 然而，直到 2011 年 HTML5 和 CSS3 程式碼的引入，這項技術才真正受到網頁設計師的歡迎。 支持者認為這是一種擁抱網頁流暢性的簡單方式，並將視差背景作為提升使用者參與度和改善整體體驗的工具。 不過，普渡大學在 2013 年發表的一項研究也指出：「......雖然視差捲動增強了使用者體驗的某些方面，但並不一定能改善整體的使用者體驗」。


為什麼要使用視差圖像？

    設計師嘗試使用視差圖像主要有兩個原因：

  • 協助敘述故事 (Storytelling)

    視差圖像非常適合用來講故事。好的設計在於溝通——當我們設計網站時，我們希望與訪客溝通，並講述一個關於產品、服務或品牌的動人故事。 視差捲動讓我們能夠讓網站訪客沉浸在獨特的線上體驗中。甚至可以創建一個單頁式網站 (One-page website)，讓訪客無需跳轉頁面，就能讀完關於您產品或品牌的完整故事。

  • 提升使用者參與度 (User Engagement)

    視差圖像可用於提升使用者的參與度。降低網站的「跳出率 (Bounce Rate)」是網頁設計中必須謹慎對待的目標之一。 跳出率衡量的是訪客在僅查看一頁後就點擊「上一頁」或關閉分頁的百分比。高跳出率會損害您的搜尋排名，因此任何能降低此比率的行動都是值得的。 一個精心製作的視差頁面能提供動態的體驗，往往能讓使用者在頁面上停留更長的時間。


如果您想在網站中加入視差捲動效果，建議參考以下技巧：

  • 適度使用 (Use it selectively)

    與其在所有頁面隨時隨地使用，不如將視差捲動應用在特定元素上，效果會更強大。 建議僅在頁首 (Header)、標題區，或者只在首頁使用。您會希望視差捲動能強化網站的設計感，而不是分散使用者的注意力。

  • 影像壓縮 (Image compression)

    視差捲動使用各種媒體檔案和 CSS 形狀來創造景深感。 您一定不希望因為圖檔過大導致載入緩慢，進而犧牲了網頁效能與使用者體驗。

  • 善用色彩搭配 (Play with color)

    視差捲動不僅僅是關於前景和背景的圖像，色彩也是關鍵。 您可以嘗試在背景使用飽和度較低的色調，而在前景使用更鮮豔活潑的顏色，以創造出更強烈的層次感與深度。
