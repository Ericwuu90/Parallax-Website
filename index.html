视差滚动特效

作者：Vineet Kumar

什么是视差滚动 (Parallax Scrolling)？

视差滚动是一种网页设计技术，让网页背景的移动速度比前景慢。当访客向下滚动页面时，这会产生 3D 立体效果，增加景深感并创造更身临其境的浏览体验。 视差是基于视觉错觉的原理。由于人眼会觉得靠近我们的物体比远处的物体大，我们也会感知到远处的物体移动得较慢。 这种错觉早已被应用于各种媒体中以营造真实感。它最早用于传统动画，可追溯到迪士尼的《白雪公主与七个小矮人》，以及像《超级马里奥》这样的经典电子游戏。 随着 CSS 和 HTML 的进步，视差效果后来演变成了我们今天所熟知的网页设计形式。

早在 2007 年，网页开发者 Glutnix 就在博客文章中分享了浏览器端视差滚动的早期实现，其中包含了示例代码以及使用 JavaScript 和 CSS 2 制作的 Demo，当时甚至还支持 Internet Explorer 6。 然而，直到 2011 年 HTML5 和 CSS3 代码的引入，这项技术才真正受到网页设计师的欢迎。 支持者认为这是一种拥抱网页流畅性的简单方式，并将视差背景作为提升用户参与度和改善整体体验的工具。 不过，普渡大学在 2013 年发表的一项研究也指出：「......虽然视差滚动增强了用户体验的某些方面，但并不一定能改善整体的用户体验」。


为什么要使用视差图像？

    设计师尝试使用视差图像主要有两个原因：

  • 协助叙述故事 (Storytelling)

    视差图像非常适合用来讲故事。好的设计在于沟通——当我们设计网站时，我们希望与访客沟通，并讲述一个关于产品、服务或品牌的动人故事。 视差滚动让我们能够让网站访客沉浸在独特的线上体验中。甚至可以创建一个单页式网站 (One-page website)，让访客无需跳转页面，就能读完关于您产品或品牌的完整故事。

  • 提升用户参与度 (User Engagement)

    视差图像可用于提升用户的参与度。降低网站的「跳出率 (Bounce Rate)」是网页设计中必须谨慎对待的目标之一。 跳出率衡量的是访客在仅查看一页后就点击「后退」或关闭标签页的百分比。高跳出率会损害您的搜索排名，因此任何能降低此比率的行动都是值得的。 一个精心制作的视差页面能提供动态的体验，往往能让用户在页面上停留更长的时间。


如果您想在网站中加入视差滚动效果，建议参考以下技巧：

  • 适度使用 (Use it selectively)

    与其在所有页面随时随地使用，不如将视差滚动应用在特定元素上，效果会更强大。 建议仅在页头 (Header)、标题区，或者只在首页使用。您会希望视差滚动能强化网站的设计感，而不是分散用户的注意力。

  • 图像压缩 (Image compression)

    视差滚动使用各种媒体文件和 CSS 形状来创造景深感。 您一定不希望因为图档过大导致加载缓慢，进而牺牲了网页性能与用户体验。

  • 善用色彩搭配 (Play with color)

    视差滚动不仅仅是关于前景和背景的图像，色彩也是关键。 您可以尝试在背景使用饱和度较低的色调，而在前景使用更鲜艳活泼的颜色，以创造出更强烈的层次感与深度。
